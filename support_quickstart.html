QuickStart
This guide will get you started using the Sonic Adventure Homepage.

Introduction to the Browser
The Sonic Adventure Homepages are seen using the Browser included with Sonic Adventure. You're using it now! You can use the analog pad to move the cursor on the screen and use the A button to click on any links.

Command Cluster
Hitting the start button brings up the command cluster. The command cluster consists of nine (9) buttons: Reload, Mail, Address, Back, Home, Forward, Return to Game, Stop and Options. To exit the command cluster, hit the start button again.

Reload
Refreshes the current page. Click this to get new content from the server.

Mail
Check your email by clicking here.

Address
Keeps track of important email addresses. Send mail to them by clicking here.

Back
Go to previous page.

Home
Takes you to the Sonic Adventure Homepage.

Forward
Go to next page. You can only use this button after you hit the back button at least once.

Return to Game
Use this to go back to the Game.

Stop
This button cancels any ongoing commands.

Options
You can change your ISP, dial up and browser settings here.

 
About the Sonic Adventure Homepage
The Sonic Adventure Homepage is a closed network all about Sonic Adventure. Here, you can participate in World Rankings, Events, Chao Gardens, and so much more! Also, we have a community where you can make friends and chat about Sonic Adventure.
