Intelligent questions indeed! Okay, Here are your answers:

1. SWIMMING > POWER explanation
Basically, each animal that you can rescue from the Sonic Adventure game can be classified into specific character traits. For example, bunnies and deer share RUNNING qualities where elephants and lions share POWER qualities. If you give your Chao one specific characteristic enough times, it will evolve into a shape that would be better at that characteristic! Eventually, it will cocoon and emerge as an adult with even more distinct changes depending on characteristics! Now, you can opt to give it more of the same characteristic, or change characteristics to get a Combo Chao! The example above shows you how to create a Sumo Chao!

2. VMU File Management
Okay, looks like you have 93 blocks left on your VMU! You are getting a COPY FAILED error putting a Chao in your VMU because you need more space! You will have to delete a file to make enough for Chao Adventure. First, let's identify which files are which:

SONICADV_INT  (10 blocks)
This is your main save file. Don't delete this!

SONICADV_ALF  (28 blocks)
This is your A-Life File (ALF, get it?). This contains all the information for all your Chao in all your Chao Gardens. Don't delete this either!

SONICADV_506  (68 blocks)
This is a Minigame Event Download file. This particular one is for Sonic2k: Treasure Hunt. You can delete this file after you are finished competing. If you want to download a Holiday Event File, you need to delete this as well. Only one Event File at a time can be stored.

SONICADV_H07  (11 blocks)
This is your World Rankings Upload file! This file was created when you pushed the secret handshake at the file select menu (Y+A). You can delete this after you have uploaded your time to World Rankings.

Looking at all your files, it seems most logical to erase the SONICADV_506 file in order to put your Chao in your VMU. You can always download the Event Minigame from the website when you are finished. If your Chao like Knuckles, just choose Knuckles from the Adventure menu and you can still have him take care of your Chao.
